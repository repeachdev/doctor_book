# 단권화 요약본 — 디자인 가이드

원본 교재(「KMLE·ACM 단권화 정리본」)의 **레이아웃·서식·디자인 요소만** 추출해
요약본 HTML에 그대로 적용하기 위한 가이드입니다. 실제 구현은 [`template.html`](./template.html)에
모두 들어 있으며, 이 문서는 그 사용법과 규칙을 설명합니다.

> **원칙**
> - 브랜딩 제거(중립): 유니온 / UNION Books / HI-PASS / unionbook.co.kr 표기·로고는 **전부 삭제**.
> - 연도: `2025 대비` → **`2027 대비`** 로만 변경(내용 업데이트 아님).
> - 판형: **A4** (인쇄/PDF 저장 기준). 화면 미리보기 시 회색 여백·안내문은 인쇄에서 사라짐.
> - 요약 강도: 핵심 압축(원본의 1/2~1/3). 이 가이드는 **서식**만 다루며 내용 요약은 별도 작업.

## 1. 페이지 골격

한 페이지 = `<section class="page c-purple"> … </section>` 한 블록.
- A4(210×297mm), 안쪽 여백 약 14/15/16/15mm.
- 구조 순서: **러닝헤더 띠 → (장 탭) → 절 뱃지 → 본문 → 페이지 푸터**.
- 푸터는 `margin-top:auto` 로 항상 페이지 바닥에 붙습니다.
- 새 페이지는 이 블록을 통째로 복제하고 안쪽만 교체.

### 페이지 채우기 — 원칙 하나
이 템플릿은 **수동 페이지 모델**입니다(자동 흐름 아님): `<section class="page">` 한 칸이 A4 한 장이고, footer의 페이지 번호도 직접 적습니다.
- `.page{min-height:297mm}` 라서 내용이 적으면 footer가 페이지 바닥에 붙고, 하단에 공백이 남는 건 정상입니다.
- **원칙: 한 섹션엔 A4 한 장 분량까지만.** 넘치면 섹션이 늘어나 footer가 다음 장으로 밀립니다 — 그때만 그 섹션을 둘로 나누면 됩니다.
- 어디서 끊을지는 상식대로: 표·그림·콜아웃은 페이지 중간에서 자르지 말고, 남은 공간에 안 들어가면 다음 섹션으로 넘깁니다.
- (자동 확인용) 렌더한 PDF 페이지 수가 섹션 수와 같으면, 넘친 섹션이 없다는 뜻입니다. 더 많으면 어떤 섹션이 한 장을 초과한 것이니 그 섹션만 나눕니다.

### 장(章) 컬러
페이지(또는 개별 요소)에 컬러 클래스를 붙이면 뱃지·탭·표헤더·소제목 바 색이 일괄 변경됩니다.

| 클래스 | 색 | 권장 용도(예) |
|---|---|---|
| `.c-purple` | 보라 `#5f4b9c` | 기본 / 순환기 |
| `.c-teal` | 틸 `#277f74` | 호흡기 등 (흰 글씨 대비 AA) |
| `.c-blue` | 블루 `#3f6cc0` | 소화기 등 |
| `.c-orange` | 오렌지 `#d9772f` | 강조 구획 |
| `.c-red` | 레드 `#c0463f` | 강조 구획 |

원본은 목차에서 장마다 색을 달리하므로, 권/장 구성에 맞춰 색을 배정하세요.

## 3. 타이포그래피

- 한글 본문: Pretendard → Malgun Gothic(맑은 고딕) 폴백. (설치 폰트 없어도 무난히 렌더)
- 영문 의학용어: `.eng` (이탤릭). 원본도 영문 학명/약어를 이탤릭·괄호로 병기.
- 본문 10.3pt / line-height 1.5. 표 9.4pt, 콜아웃 9.8pt.
- 크기 위계: 절 제목 13pt → 번호 소제목 11pt → 밑줄 소제목 10.4pt → 본문 10.3pt.

### 영어 의학용어 표기 규칙
요약은 **서식**이 아니라 **내용** 영역이지만, 자주 어기므로 여기 명시:
- 교재에 **영어로만** 된 용어를 한국어로 옮길 때 → **억지 번역하지 말고 영어 그대로**. 예: `Kussmaul's sign` → `Kussmaul sign` (❌ `Kussmaul 징후`). 생물·의학 용어는 영어로 공부하기 때문.
- 한글 뜻을 붙일 거면 → **`English (한국어)`** 순서. 예: `fixed splitting (고정 분열)`, `Holosystolic (전수축기)`.
- 교재에 **이미 있는** `한국어(영어)` / `영어(한국어)` 병기는 **그대로 둠**(변경 금지).
- 이 규칙은 **원본이 영어-only인 것을 한국어로 옮길 때만** 적용.

## 4. 컴포넌트 사용법

`template.html` 의 두 번째 페이지("컴포넌트 모음")가 복붙용 레퍼런스입니다.

### 4.1 러닝헤더 띠 `.runhead`
```html
<header class="runhead">
  <span class="year">2027 대비</span>
  <span class="connector short"></span>
  <span class="title"><b>KMLE</b>· ACM 단권화 요약본</span>
  <span class="connector"></span>
</header>
<div class="chaptab">01 순환기</div>
```
- `.year` 회색 알약 = `2027 대비`(고정).
- `.title` 중앙 흰 알약 = 교재명(중립). **`HI-PASS`/유니온 표기 금지.**
- `.chaptab` 우측 상단 탭 = `장번호 + 장이름`, 장 컬러로 표시.

### 4.2 절 번호 뱃지 `.sec`
```html
<h2 class="sec"><span class="num">03</span><span class="label">판막질환</span></h2>
```
컬러 라운드 사각 뱃지 + 절 제목. 번호는 두 자리(`01`, `02` …). **시맨틱 제목 태그**(`h2`/`h3`/`h4`)를 쓰면 PDF 북마크·스크린리더 목차가 생깁니다: 절=`h2.sec`, 번호 소제목=`h3.h-num`, 밑줄 소제목=`h4.h-sub`.

### 4.3 소제목
- 번호 소제목(옅은 바 + 좌측 틱): `<h3 class="h-num">1. 대동맥판막협착증 <span class="eng">(Aortic stenosis, AS)</span></h3>`
- 밑줄형 소제목(표/세부 묶음 앞): `<h4 class="h-sub">중증도 분류</h4>`

### 4.4 불릿 계층 `ul.bul`
중첩 `<ul>` 로 단계 표현. 마커 자동: 1단계 `●`(컬러), 2단계 `○`, 3단계 `–`.
- 항목 변형 클래스: `li.dash`(–) · `li.note`(※ 빨강) · `li.arrow`(▶ 컬러) · `li.plain`(마커 없음, `(1)`·`①` 번호항목용).

### 4.5 인라인 강조
`.r`(빨강) · `.b`(파랑) · `<mark class="k">`(KMLE 형광) · `<mark class="a">`(ACM 음영) · `.eng`(이탤릭 영문) · `.u`(밑줄).

### 4.6 기출 콜아웃 `.callout`
```html
<div class="callout"><span class="tag">KMLE 기출</span> … 문제 요지 …</div>
```
좌측 컬러바 + 옅은 배경. 태그는 `KMLE 기출` / `ACM 기출`.
범례 줄(`.legend`)로 형광=KMLE, 음영=ACM 을 안내할 수 있음.

### 4.7 표 `table.tbl`
- `thead th` = 장 컬러 헤더(흰 글씨).
- `tbody th` = 좌측 라벨 열(옅은 컬러 배경).
- 숫자 칸은 `td.center`. 짝수 행 옅은 음영 자동.

### 4.8 페이지 푸터 `.pagefoot`
```html
<footer class="pagefoot"><span class="pno">12</span><span class="dots"></span></footer>
```
회색 페이지번호 알약 + 점선 리더. 항상 페이지 하단.

### 4.9 그림 `.figwrap`
```html
<figure class="figwrap">
  <img src="figs/fig_auscultation_areas.png" alt="판막별 청진 부위">
  <figcaption>판막별 청진 부위</figcaption>
</figure>
```
- 교재 그림을 PyMuPDF로 캡처해 넣습니다. **검은/회색 테두리 없음**, 풀폭(`max-width:100%`).
- 캡션은 그림 설명만. **`· 본 교재 p.X 발췌` 같은 출처 표기는 넣지 않습니다.**
- 그림은 가로로 넓은 경우가 많아 **표/본문 옆 좁은 칼럼에 두지 말고 단독 행**으로 크게 배치(축소 금지).
- 캡처 방법: 원본 PDF는 페이지 통째 스캔이라 그림을 객체로 추출할 수 없음 → `page.get_pixmap(clip=Rect)` 로 **그림+라벨 전체를 넉넉히 잡고 → 크롭본을 직접 보며 박스 조정**.

## 5. 인쇄 / PDF

- `@media print` 에서 회색 배경·그림자·안내문 제거, `@page size:A4; margin:0`, 페이지마다 `page-break-after`.
- 전역 `print-color-adjust:exact` 가 걸려 있어 **표 헤더·뱃지·형광/음영 색이 인쇄에서 자동 보존**됩니다(예전엔 "배경 그래픽" 수동 체크에 의존). Firefox 등 일부 브라우저는 여전히 "배경 그래픽"을 켜야 할 수 있으니, 색이 빠지면 그 옵션을 확인하세요.
- 표·콜아웃·절 제목에 `break-inside:avoid`/`break-after:avoid` 가 걸려 있어 페이지 경계에서 잘리지 않고 다음 장으로 넘어갑니다.
