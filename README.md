# 한국의사국가시험 단권화 마인드맵

유니온 HI-PASS KMLE · ACM 단권화 정리본의 목차를 인터랙티브 마인드맵으로 정리한 프로젝트입니다.

## 보기

- 마인드맵: [`index.html`](./index.html) — 가로형 마인드맵, 카드 클릭 시 펼침/접힘, 피그마식 확대·축소·이동
- GitHub Pages 배포 시 루트 접속하면 바로 마인드맵이 열립니다.

## 구성

| 경로 | 내용 |
|---|---|
| `1/` | 1권 (순환기 · 호흡기 · 소화기) — 요약본 작성중 |
| `2/` | 2권 (신장 · 내분비 · 감염 · 류마티스 · 알레르기 · 혈액) — 요약본 작성중 |
| `3/` | 3권 (외과총론 · 외과각론) — 요약본 작성중 |
| `index.html` | 인터랙티브 마인드맵 (사이트 진입 페이지) |
| `DESIGN.md` | 디자인 시스템 정의 (Mistral AI) |
