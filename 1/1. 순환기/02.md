# 순환기 (Cardiology)

## 부정맥 (Arrhythmia)

> **강의 들어가며** — 부정맥은 ① 먼저 **항부정맥약(Vaughan-Williams 분류)**과 진단 도구를 깔고 → ② 동기능부전(서맥) → ③ 조기수축(APB·VPB) → ④ 상심실성 빈맥(심방조동·심방세동·PSVT·WPW) → ⑤ 방실차단 → ⑥ 심실성 빈맥(VT·VF) → ⑦ 특수 파형(각차단·디지탈리스 중독·고칼륨혈증)과 ⑧ 급사 3대 질환(Brugada·Long QT·HCMP)·Torsades 순으로 봅니다. 시험은 "어떤 ECG 파형/임상상황 → 무슨 부정맥, 무슨 치료(특히 **약물 선택과 금기**)"를 묻는 매칭이 압도적입니다. 약물 금기(WPW+AF에서 AV node 차단제, Long QT에서 QT 연장 약물)가 결정적 출제 포인트입니다.

---

### 01. 항부정맥약 종류 (Vaughan-Williams 분류)

> 💡 우리 시험에서는 **Class II, IV**를 주로 출제하고, **Class I, III는 거의 출제되지 않았습니다.**

**1) Class I : Na channel blocker**
- (1) IA (QT 간격 ↑) : Quinidine, Procainamide, Disopyramide
- (2) IB (불응기 ↓) : Lidocaine, Phenytoin, Mexiletine
- (3) IC (QT 간격 ↑) : Propafenone, Flecainide, Moracizine

**2) Class II : β-blocker (심박수 ↓)** → Propranolol, Nadolol, Esmolol

**3) Class III : K channel blocker (불응기 ↑)** → Sotalol, Amiodarone, Bretylium, Ibutilide, Dofetilide

**4) Class IV : Ca channel blocker (심박수 ↓)** → (노란 형광펜)**Non-DHP (Verapamil, Diltiazem)**

**5) Others** : (노란 형광펜)**Adenosine (작용시간 매우 짧다!)**, Digitalis (작용시간 매우 길다!)
- (1) IA, IC : 전도속도 느리게
- (2) IA, II : 불응기 길게
- (3) IB : 불응기 짧게
- (4) II, IV : HR 느리게 (SA node, AV node에 작용)

### 02. 시기에 따른 부정맥의 진단 방법

> 💡 발생 빈도에 맞춰 검사를 고릅니다 : **24시간 홀터(하루에도 여러 번) → 활동사건 기록기(1주일 1~2회) → 삽입형 기록기(1년에 몇 번)**.

| 진단 방법 | 적응증 |
|---|---|
| (노란 형광펜)**24hr Holter monitoring** | 하루에도 여러 번 부정맥이 발생하는 경우 (1~2일간 관찰) |
| Event recorder | 1주일에 1~2번 부정맥이 나타나는 경우 (2~4주간 관찰) |
| Implantable event recorder | 1년에 1~2번 부정맥 또는 원인 모를 실신이 반복되는 경우 / 심실부정맥 여부를 판단하려는 목적 (14개월간 관찰) |
| Treadmill exercise test | 안정형 협심증의 진단검사 |
| Head up tilt test | 미주신경성 실신, 기립성 저혈압의 진단검사 |
| Electrophysiologic study | 위의 방법으로 진단이 어려운 경우 고려 |

### 03. 동기능 정상 / 동기능부전 (Sinus node dysfunction)

**1) 동성빈맥 (Sinus tachycardia)** : Heart rate > 100/min
- (1) 심전도 : QRS complex 앞에 P파가 항상 존재. Sinus tachycardia로 심박수가 올라가면 **앞 QRS complex의 T파와 뒤 QRS complex의 P파가 중첩되어 마치 하나의 P 또는 T파처럼 보일 수 있어** 주의해야 함
- (2) 원인 : 대개 이차적인 원인에 대한 생리적 반응으로 발생 — (노란 형광펜)**갑상샘중독증**, fever, exercise, volume depletion 등
- (3) 치료 : 증상 있을 시 원인 교정을 통해 치료. 빈맥으로 인한 증상 발생 시 β-blocker 투여를 고려

**2) 동성서맥 (Sinus bradycardia)** : Heart rate < 60/min
- (1) 심전도 : 정상 P파와 QRS complex가 regular한 형태로 (느리게) 존재
- (2) 원인 : β-blocker 사용, 고령, hypothyroidism, typhoid fever(relative bradycardia) 등. (노란 형광펜)**RCA를 침범한 acute coronary syndrome에서 SA nodal artery ischemia에 의한 sinus bradycardia**가 발생할 수 있음
- (3) 치료 : 증상 없으면 경과 관찰. 증상 있을 시 기저질환 치료 or (노란 형광펜)**일시적 심박수 상승제(ex. atropine)** 투여

**3) 동정지 (Sinus arrest)** : 일정 시간 동안 동방결절에서 적절한 자극을 발생시키지 못해 → 잠깐 동안 심방·심실의 박동(P파·QRS군)이 없음

**4) 동방차단 (SA node exit block)** : 동방결절은 정상적으로 자극을 발생시키나, 동방결절-심방 사이 전도 장애로 자극이 심방에 전도되지 못하는 경우

> 💡 **동성서맥에 β-blocker는 금기**입니다. 증상성 서맥에서 심박수를 올릴 때는 **atropine(또는 isoprenaline)**을 씁니다.

#### 동기능부전증후군 (Sick sinus syndrome, SSS) / 서맥-빈맥 증후군

- 동성서맥 또는 동정지가 발생하면서 (노란 형광펜)**실신(syncope), 어지러움(dizziness), 졸도(near syncope), confusion, 피로(fatigue)·CHF**가 발생할 수 있음.
- (빨간 형광펜)**서맥-빈맥 증후군 (tachycardia-bradycardia syndrome)** : 빈맥과 서맥이 교대로 나타나는 것. 빈맥(심방세동·심방조동 등)이 끝난 뒤 동결절이 정상 박동을 회복하지 못해 긴 휴지기(pause)가 생기고, 이 시기에 실신·어지러움이 나타남.

**1) 임상양상** : 호흡곤란, 전실신(presyncope), 실신

**2) 진단** : Surface ECG에서 동성서맥·sinus exit block 등을 확인. 휴지기 PP interval이 매우 길게 나타남(secondary pacemaker 부재 시). Holter monitoring으로 sinus arrest 확인.

> 🖼 [그림필요 · p16] 서맥-빈맥 증후군 ECG (긴 휴지기/pause 5.4초, sinus arrest 후 secondary pacemaker 결여) — (미확보: 원서 p16 ECG strip 스캔 필요)

**3) 치료**
- (1) 증상이 없는 경우 : 치료 불필요, 경과관찰
- (2) 증상이 있는 경우
  - ① (노란 형광펜)**원인 약제 중단** : 일부에서는 BB(m/c cause), CCB, cardiac glycoside 등 심혈관계 약제 사용 시에만 SSS가 나타날 수 있어, 원인으로 의심되는 drug를 중단한 후 반응을 관찰
  - ② (빨간 형광펜)**영구심박동조율기(permanent pacemaker)** (단, 급성기 치료에는 임시심박동조율기)
  - ※ 동기능부전증후군 환자는 동방결절 이외의 이상도 동반되는 경우가 많아, atrial pacing 이외에 ventricular pacing까지 가능한 (노란 형광펜)**dual chamber pacemaker**를 삽입해야 함
- (3) marked sinus bradycardia : (노란 형광펜)**atropine, isoprenaline**

### 04. 심방조기수축 (Atrial premature beat, APB)

**1) 특징**
- (1) 정상인에서도 관찰될 수 있는 양성 부정맥 중 흔함
- (2) 술·담배, 교감신경 항진제 등에 의해 유발 가능
- (3) (노란 형광펜)**심장질환 환자에서 심방세동·심방조동·PSVT 등 빈맥성 부정맥을 야기**할 수 있음

**2) 증상** : 대개 무증상. 간혹 "가슴이 덜컹거리는 느낌" 호소

**3) 심전도**
- (1) (빨간 형광펜)**Early abnormal P wave** : 심방의 이소성 중추(ectopic focus)에서 조기 심방자극이 예상보다 빨리 나타나는, sinus rhythm과는 다른 이상한 P파
- (2) (노란 형광펜)**비대상성 휴지기 (non-compensatory pause)** : APB 앞뒤로 PP interval이 정상 PP interval의 2배가 되지 않음 (APB가 SA node로 들어가 reset시키기 때문)

> 🖼 [그림필요 · p17] Ectopic P wave를 동반한 APB ECG — (미확보: 원서 p17 ECG strip 스캔 필요)

**4) 치료**
- (1) 증상(-) : 치료 필요 없음(안심시키기가 중요) → 24시간 홀터검사 권유
- (2) 증상(+) 또는 PSVT 시 : 유발인자 제거, β-blocker 또는 mild sedation

### 05. 심실조기수축 (Ventricular premature beat, VPB)

**1) 원인**
- (1) 교감신경 항진의 결과
- (2) 심근경색
- (3) 저산소증
- (4) 전해질 이상 (특히 저칼륨혈증)
- (5) 다른 심질환의 결과
- (6) 특별한 질환이 없는 경우에도 정상적으로 나타날 수 있음

**2) 증상**
- (1) 대개는 증상이나 기능상의 이상을 유발하지 않음
- (2) frequent PVC의 경우 (노란 형광펜)**좌심실 기능의 저하**를 유발할 수 있음
- (3) 두근거림, 실신, 어지러움
- (4) (빨간 형광펜)**Cannon a wave** (심방·심실 수축의 dissociation → 삼첨판(TV)이 닫힌 상태에서 심방이 수축)

**3) 심전도 소견**
- (1) P파의 선행 없이 발생하는 (빨간 형광펜)**넓고 이상한 모양의 QRS군 (wide, bizarre, abnormal QRS complex)**
- (2) Fixed coupled VPB : 선행하는 정상 QRS군과 일정한 간격(coupling interval)으로 발생
- (3) (빨간 형광펜)**대상성 휴지기 (compensatory pause)** : VPC가 끼어도 SA node 박동에 영향을 주지 않아, VPB 직후 약간의 휴지 후 정상 sinus 박동이 그대로 돌아옴 (VPB가 SA node를 reset시키지 않기 때문)
- (4) LV origin이면 RBBB pattern, RV origin이면 LBBB pattern
- (5) 심실성 이단맥(ventricular bigeminy) : 정상 QRS와 VPB가 1:1 교대
- (6) 심실성 삼단맥(ventricular trigeminy) : 정상 2회마다 VPB 1회
- (7) 3개 이상의 VPB가 연속되고 HR이 분당 100회 이상 → (노란 형광펜)**심실빈맥(Ventricular Tachycardia)**

> 🖼 [그림필요 · p18] VPB ECG (wide QRS, compensatory pause 표시) — (미확보: 원서 p18 ECG strip 스캔 필요)

**4) 치료**
- (1) 무증상 : 치료 필요 없음
- (2) 증상(+) : β-blocker가 1차 약제. non-DHP CCB, Class I·III 항부정맥제도 사용 가능 (심한 경우 RFCA(catheter ablation)도 고려)
- (3) 구조적 심질환이 있는 경우
  - ① 잦은 VPC가 sudden/non-sudden cardiac death 증가와 관련 있다고 알려져 있으나, 직접적인 원인-결과 관계는 아님
  - ② (빨간 형광펜)**항부정맥제 치료는 권장하지 않음** : proarrhythmic effect로 인해 오히려 사망을 증가시킬 수 있음

### 06. 심방조동 (Atrial flutter)

**1) 정의** : 심방 내 re-entry circuit, 혹은 ectopic pacemaker로 인해 발생하는 비교적 규칙적이고 빠른 심방의 흥분

**2) 특징**
- (1) 유발인자 : (빨간 형광펜)**개흉수술**, 구조적 심질환 등 (심장 내부 electrical system의 장애)
- (2) 심전도 : P파가 (노란 형광펜)**톱니 모양(sawtooth pattern, F파)으로 분당 240~300회**의 규칙적인 심방파
- (3) 심실은 방실전도 차단에 의해 (노란 형광펜)**75~150bpm**
- (4) PSVT와 구별이 되지 않을 때는 adenosine을 주입해 증상이 호전되면 PSVT로 진단

**3) 치료**
- (1) (빨간 형광펜)**심방세동(atrial fibrillation)에 준해서 치료**
- (2) DC cardioversion 시행 시 낮은 에너지 (노란 형광펜)**50~100J**

---

### 07. 심방세동 (Atrial fibrillation, AF)

> 💡 (빨간 형광펜)**MS, 노인에서 흔하다!** — 빈출.

**1) 심전도** : P파를 찾기 어렵고 PR 간격이 불규칙하면서 narrow QRS군이 보임 (P파와 QRS complex가 모두 불규칙한 (빨간 형광펜)**Irregularly irregular pattern**)

> 🖼 [그림필요 · p19] AF ECG (P파 소실, 불규칙한 RR 간격, fibrillation wave) 및 AF with rapid ventricular response(AF with RVR) — (미확보: 원서 p19 ECG strip 스캔 필요)

**2) 증상** : 대부분 무증상이나 두근거림, 빈맥으로 인해 심실이 충분히 이완되지 못해 체내의 산소 요구량을 따라가지 못하여 호흡곤란 동반 가능

**3) 검사** : EKG상 AF가 관찰되는 경우, 심장의 구조적 문제로 AF가 생겼는지 확인하기 위해 (노란 형광펜)**심초음파**를 시행

**4) Acute AF의 치료**

**(1) 급성기 심박수 조절 (Rate control)** — 좌심실 수축기능(EF)에 따라 약을 고름
- ① 좌심실 수축기능 (빨간 형광펜)**40% 이상** : β-blocker(bisoprolol, propranolol) and/or non-DHP CCB(diltiazem, verapamil) + 필요시 디곡신
- ② 좌심실 수축기능 (빨간 형광펜)**40% 이하** : β-blocker + 필요시 디곡신 (← non-DHP CCB 제외)

> 🖼 [그림필요 · p20] 급성기 심방세동의 율동조절(Rhythm control) 알고리즘 (전기적 동율전환 / Amiodarone·Sotalol / Flecainide·Propafenone / "pill in the pocket") — (미확보: 원서 p20 흐름도 스캔 필요)

**(2) 항응고제 (Anticoagulation)** : 혈전색전증 위험도를 (빨간 형광펜)**CHA₂DS₂-VASc score**로 평가하고 항응고제 사용 여부를 결정함
- ① CHA₂DS₂-VASc score : **C**ardiac failure, **H**TN, **A**ge ≥75, **D**M, **S**troke, **V**ascular disease, **A**ge 65~74, **S**ex category(female)

| | CHA₂DS₂-VASc score | 점수 |
|---|---|:---:|
| C | 심부전 : 심부전에 의한 증상/징후가 있거나 좌심실 구출률이 40% 이하로 저하된 경우 | 1 |
| H | 고혈압 : 안정 시 2회 이상 측정한 혈압 > 140/90mmHg 또는 현재 항고혈압 약제를 복용 중인 경우 | 1 |
| A₂ | 연령 : 75세 이상 | 2 |

| S₂ | 뇌졸중, 일과성 허혈성 발작, 전신 색전증의 과거력 (4mm 이상의 두께 또는 궤양성 또는 유동성 죽상반을 의미) | 2 |
| V | 혈관질환 : 심근경색, 말초혈관질환, 대동맥 죽상반의 과거력 | 1 |
| A | 연령 : 65~74세 | 1 |
| Sc | 여성 | 1 |

- ② CHA₂DS₂-VASc score에 따른 항응고제 사용

| 점수 | 항응고제 사용 |
|---|---|
| 0 | 항응고제를 사용하지 않아도 됨 |
| 1 | 사용하지 않거나 Warfarin 혹은 NOAC 중에서 선택 가능 |
| 2 (여성은 3점) | (빨간 형광펜)**Warfarin(INR 2.0~3.0)**, NOAC(dabigatran, rivaroxaban, apixaban, edoxaban) |

- ※ NOAC의 장점 : Therapeutic range(INR)가 넓어서 dose 조절이 쉬움 (정상 INR은 2~3 정도)
- ※ Warfarin : Vit. K antagonist로 INR 2~3을 목표로 시행. (빨간 볼드 밑줄)**MV stenosis, 인공 판막, 심한 신부전에선 NOAC 대신 Warfarin만 사용 가능**

- ③ NOAC 적응증

| 기계판막치환술 | 금기 |
| 중등도 이상의 승모판 협착증 | 금기 |
| 경도·중등도의 다른 판막 질환 (경도/중등도 대동맥판막 협착증·역류증 또는 퇴행성 승모판 역류증) | 적합 |
| 심한 대동맥판막 협착증 | 데이터 부족 |

| 승모판 성형술 (수술 3개월 이후) | 적합 |

| 경피적 대동맥 판막 치환술(TAVI) | 적합 |
| 비후성 심근병증 | 적합 |

- ④ AF가 발생한 지 24~48시간 동안 적절한 항응고요법을 시행하지 못한 경우 : (노란 형광펜)**Trans-esophageal echocardiography(TEE)로 LA thrombus 존재 여부를 확인**

**5) Chronic AF (지속시간 7일 이상)의 치료** : 만성 심방세동은 그 자체로 심방 내 기질적 변화를 초래하여 더욱 만성화하므로, 억지로 율동전환을 하기보다 (빨간 볼드 밑줄)**"rate control + anticoagulation"을 치료원칙으로 삼는다.**
- (1) Rate control : (노란 형광펜)**β-blocker(bisoprolol, propranolol)**, non-DHP CCB(verapamil, diltiazem), digoxin
- (2) Stroke prevention (Anticoagulation; Warfarin or NOAC)의 적응증
  - ① CHA₂DS₂-VASc score 2점 이상
  - ② 심방세동이 48시간 이상 지속되거나 AF duration을 정확히 알 수 없는 경우
  - ③ 이전에 cardioversion 받았던 경우
  - ④ 뇌졸중(stroke)의 과거력
- (3) 비약물적 치료 : (노란 형광펜)**RF catheter ablation** 또는 surgical ablation therapy

> 💡 증상이 심한 심방세동의 경우 고주파카테터절제술(RFCA)을 고려할 수 있습니다.

**6) 갑상선기능항진증과 AF** : 갑상샘 호르몬은 심방 세포의 활동전위 기간(APD)을 단축시켜 회귀가 잘 일어나므로, (노란 형광펜)**갑상선기능항진증**에서는 (빨간 형광펜)**심방세동 또는 빈맥(sinus tachycardia)**이 발생할 수 있다. 원인질환을 치료해야 하므로 (노란 형광펜)**메티마졸(MTZ)**, 프로필티오우라실(PTU) 등을 사용.

**7) MS와 AF** : (빨간 형광펜)**Mitral stenosis(MS)**의 합병증으로 (빨간 형광펜)**심방세동**이 생길 수 있다. 증가된 제1심음과 확장기 심잡음(MS 시사)과 함께 '갑작스런 오른쪽 옆구리 통증과 소변 검사상 잠혈' 소견이 있으면, AF로 생긴 심방 내 thrombus가 신장 혈관에 embolism을 형성했을 가능성이 있어 (노란 형광펜)**복부 CT**로 embolism 여부를 확인해야 함.

> **🧩 이것만은 꼭! — 심방세동 요약**
> 1) Rate control (BB, non-DHP CCB, digitalis)
> 2) CHA₂DS₂-VASc score 2점 이상이면 Anticoagulation → Warfarin, NOAC(dabigatran, rivaroxaban, apixaban, edoxaban)
> 3) Rhythm control (구조적 심질환 유무에 따른 Amiodarone 등의 약물)
> 4) 혈역학적으로 불안정한 경우 DC cardioversion
> 5) 약물치료 실패 시 유증상 심방세동에 대해 RFCA 가능

### 08. 방실차단 (AV block)

**1) 1도 방실차단 (1° AV block)**
- (1) 심전도 : 비교적 균일한 sinus rhythm. (빨간 형광펜)**PR 간격이 정상(0.12~0.20s)보다 늘어남**. QRS군 모양은 비교적 정상
- (2) 치료 : 대부분 무증상

**2) 2도 방실차단, Mobitz type I (Wenckebach)**
- (1) 심전도 : PR 간격이 점점 길어지다가 갑자기 한 박동의 QRS군이 소실 (P파만 보임)
- (2) 증상(+) : 이식심박동조율기(pacemaker)
- (3) 증상(-) : 경과관찰

**3) 2도 방실차단, Mobitz type II**
- (1) 심전도 : PR 간격이 일정하다가 갑자기 QRS군 소실
- (2) 원인 : 구조적 손상에 의한 전도장애
- (3) 치료 : (빨간 형광펜)**증상 유무와 관계없이 permanent pacemaker**

**4) 3도 방실차단 = 완전방실차단 (3° AV block)**
- (1) 심전도 : P파와 QRS군이 서로 독립적으로 발생하는 (빨간 형광펜)**방실해리** (P파 따로, QRS군 따로)
- (2) 치료 : (빨간 형광펜)**증상 유무와 관계없이 permanent pacemaker**

> 💡 AV block 유발의 흔한 원인 : 운동선수, MI, Digitalis 독성, BB/CCB 과다 사용, 급성 감염(viral myocarditis).

### 09. 발작성 상심실성 빈맥 (Paroxysmal supraventricular tachycardia, PSVT)

**1) 정의**
- (1) 기저질환 : 기저질환 없이 오는 경우(m/c), ASD, mitral valve disease, (노란 형광펜)**WPW syndrome**
- (2) 젊은 환자가 갑자기 발생한 두근거림으로 내원했을 때 가장 먼저 (빨간 형광펜)**PSVT와 조기박동(APB, VPB)** 두 가지를 의심
- (3) (빨간 형광펜)**PSVT는 두근거림이 갑자기 시작해서 갑자기 끝남.** 조기박동은 가슴이 덜컹 내려앉는 느낌이라고 표현

**2) 종류**

> 💡 KMLE에서는 AVNRT/AVRT 종류를 구분하는 문제로 출제된 적이 없습니다.

- (1) **AVNRT**
  - narrow QRS regular tachycardia (P파는 대개 거의 보이지 않음)
  - AV node 내에 slow pathway·fast pathway가 모두 존재 → AV node 내 회귀(reentry). slow pathway로 오는 신호는 대개 전달되지 않으나 APC 등으로 fast pathway가 불응기일 때 파가 전달되면 fast pathway가 거꾸로 올라가는 통로로 사용됨
- (2) **AVRT**
  - narrow QRS regular tachycardia, retrograde P파가 QRS군 뒤에 보임
  - AV node 내 통로가 아니라 외부의 accessory pathway를 통해 신호가 전달된 것 → (노란 형광펜)**WPW**
  - 💡 두근맥은 P wave가 정상, **PSVT는 absent/abnormal/inverted retrograde P wave**

> 🖼 [그림필요 · p23] AVNRT / AVRT의 회귀회로 + 각 ECG strip (narrow QRS regular tachycardia, retrograde P wave) — (미확보: 원서 p23 도식·ECG 스캔 필요)

**3) 치료**
- (1) 혈역학적 안정 시
  - ① 1st : (노란 형광펜)**Adenosine 정맥주사, 경동맥마사지/vagal maneuver**
  - ② 2nd : β-blocker, CCB
- (2) 혈역학적 불안정 시 : 직류심장동율전환(DC cardioversion 200J)

> 🖼 [그림필요 · p24] PSVT 치료 흐름도 (Regular narrow-complex tachycardia → hemodynamic instability? → No: vagal/adenosine → non-DHP CCB·β-blocker → antiarrhythmic / Yes: cardioversion → catheter ablation) — (미확보: 원서 p24 흐름도 스캔 필요)

**4) 예방** : 재발 시 (노란 형광펜)**고주파카테터절제술(RFCA)**

### 10. 조기흥분증후군 (Wolff-Parkinson-White, WPW syndrome)

**1) 기전** : 정상 전도로 이외에 (노란 형광펜)**우회로(AV bypass tract, Kent fiber)**가 존재하여, 전기신호가 회로 사이를 빠르게 전달하므로 빈맥이 발생하는 부정맥

**(1) WPW Syndrome 심전도**
- ① Short(< 0.12s) PR : accessory pathway를 통한 빠른 anterograde conduction에 의해 발생
- ② (빨간 형광펜)**Delta wave** : QRS 시작 부분의 비정상적 굴곡(slurred upstroke). AP를 타고 심실이 미리 흥분되어 발생
- ③ Wide QRS : 정상 전도 + accessory pathway를 통한 조기 흥분이 더해져 QRS가 넓어짐
- ④ Secondary ST-T wave change

> 🖼 [그림필요 · p24] WPW ECG (Delta wave, short PR interval, wide QRS) + 부전도로(accessory pathway) 모식도 — (미확보: 원서 p24 ECG·도식 스캔 필요)

**(2) WPW + AF/AFL with rapid ventricular response**
- ① (빨간 형광펜)**Fast, irregular, abnormal wide QRS complex** (💡 "FBI : Fast, Bizarre, Irregular QRS로 암기!")
- ② Polymorphic VT와의 감별
  - PMVT : baseline EKG와 상이하며, normal EKG가 거의 관찰되지 않음. 비교적 regular rhythm
  - WPW + AF : preexcitation 때의 EKG와 유사하며, normal EKG가 wide QRS 사이에서 비교적 많이 관찰됨

**3) 치료**
- (1) 증상(-) : (노란 형광펜)**경과관찰**
- (2) 항부정맥제
  - ① BB, CCB : 방실결절의 전도 ↓, 불응기 ↑
  - ② Quinidine, flecainide, propafenone : 부전도로의 전도 ↓, 불응기 ↑
- (3) 고주파카테터절제술(RFCA)
  - ① 항부정맥제 반응성이 떨어지는 재발성 부정맥에 사용
  - ② 95% 이상 성공적이며 대부분 surgical ablation보다 좋은 예후
- (4) (노란 형광펜)**WPW + PSVT** : adenosine 정맥주사, BB, CCB, 경동맥마사지
- (5) (빨간 형광펜)**WPW syndrome + Atrial fibrillation**
  - ① 혈역학적 불안정 시 : 직류심장율동전환(DC cardioversion)
  - ② 항응고제
  - ③ 혈역학적 안정 시 : (빨간 형광펜)**procainamide(class Ia) or ibutilide(class III)**
  - ④ 회복 후 반드시 : (빨간 형광펜)**고주파카테터절제법(RFCA)**
  - ⑤ (빨간 볼드 밑줄)**금기 : Verapamil, Digitalis, Lidocaine**

> 💡 WPW + AF에서는 AF의 전기신호가 부전도로를 통해 전도되므로 이 부전도로를 억제해야 한다. AV node에 작용하는 (빨간 형광펜)**digitalis·verapamil·adenosine·BB는 AV node를 억제하는 대신 부전도로의 불응기를 감소시켜** 부전도로를 통한 anterograde conduction을 증가시키고, 결과적으로 VF 발생 위험을 높인다. 따라서 WPW + AF에서 이들 약물은 금기!

### 11. 심실빈맥 (Ventricular tachycardia, VT)

**1) 정의** : 3개 이상 연속되는 심실조기수축이 분당 100회 이상의 rate로 발생되는 것

**2) 분류**
- (1) 비지속성(nonsustained) VT : 30초 이내. 특별한 원인 질환 없이도 발생하며 보통 증상이 없음
- (2) 지속성(sustained) VT : 30초 이상 지속. (빨간 형광펜)**혈역학적 불안정(저혈압)·심근허혈** 등을 초래하므로 즉시 치료 요함
- ① Sustained monomorphic VT : 주로 구조적 심질환에 의함. 각차단을 동반한 supraventricular tachycardia와 (빨간 형광펜)**방실해리(AV dissociation)**로 감별
- ② Polymorphic VT : 항상 구조적 이상을 동반하는 것은 아님. 주요 원인은 급성심근경색증·허혈성 심질환. HCMP, MVP, prolonged QT syndrome, digitalis toxicity 등에서도 유발 가능

**3) 증상** : 저혈압, 실신(흔함), 어지러움

> 🖼 [그림필요 · p26] Monomorphic VT vs Polymorphic VT ECG strip 비교 — (미확보: 원서 p26 ECG strip 스캔 필요)

**4) 심전도 진단**
- (1) (빨간 형광펜)**QRS complex > 0.14sec (wide and bizarre)** : 정상 전도시스템을 통하지 않아 속도가 느려 QRS duration이 길어짐
- (2) 심박수 100~250회/분, 대부분 R-R 간격이 규칙적 (국소적 re-entry track에 의함)
- (3) atypical RBBB pattern + left axis deviation
- (4) independent P (P파가 대부분 보이지 않음)
- (5) (빨간 형광펜)**방실해리(AV dissociation)** : QRS군은 wide·빠르고, P파는 무관하게 천천히 따로 발생
  - ① 포획박동(capture beat) : 심방의 전기신호가 전달되어 정상적인 QRS가 만들어진 경우
  - ② 융합박동(fusion beat) : 심방의 전기신호와 심실의 전기신호가 합쳐진 경우
- (6) concordance of QRS pattern (모든 흉부유도에서 QRS군 방향이 all(+) 또는 all(−))

**5) 치료**
- (1) 구조적 심질환(−), non-sustained VT
  - ① 무증상 : 경과관찰
  - ② 증상 시 : β-blocker
- (2) 구조적 심질환(+), sustained VT
  - ① 혈역학적 안정 : lidocaine, procainamide, (노란 형광펜)**amiodarone**, sotalol (반응 없거나 재발 시 DC cardioversion)
  - ② 혈역학적 불안정(저혈압·실신) : 직류 심장동율전환(DC cardioversion) 또는 overdriving pacing + ICD
- (3) sustained polymorphic VT : VF/VFL에 준하여 치료 → (빨간 형광펜)**제세동(defibrillation)**

> 🖼 [그림필요 · p28] Monomorphic VT 치료 알고리즘 (Hypotension/AMS/shock/AHF → Shock 100~360J / 안정 시 Lidocaine·Procainamide·Amiodarone) — (미확보: 원서 p28 알고리즘 스캔 필요)

**6) 예방**
- (1) (노란 형광펜)**ICD (Implantable cardioverter defibrillator)** : 구조적 심질환이 있는 환자에서 VT 재발 방지
- (2) 항부정맥제(sotalol, amiodarone) : 구조적 심질환과 동반된 life threatening polymorphic/monomorphic VT에서 1st line therapy로 사용 (단, (빨간 형광펜)**long QT syndrome으로 인한 VT가 아니어야 함**)
- (3) 고주파카테터절제술(RFCA, Catheter ablation) : 구조적 심질환이 없는 VT에서 효과적이나, 추가 심질환이 있는 경우에도 사용

### 12. 심실세동 (Ventricular fibrillation, VF)

**1) 원인** : 허혈성 심질환(AMI, cardiac death), 항부정맥 약제, 심방세동을 동반한 WPW 증후군 등

**2) 심전도** : 불규칙하고 QRS·T파를 구분할 수 없는 파형

> 🖼 [그림필요 · p28] VF ECG strip (무질서한 파형, QRS·P 구분 불가) — (미확보: 원서 p28 ECG strip 스캔 필요)

**3) 치료** : 즉시 (빨간 형광펜)**제세동(defibrillation)** & IV vasopressor(epinephrine or vasopressin)
- ※ 심실세동이나 맥박이 잡히지 않는 심실빈맥은 즉시 제세동을 하고, 제세동기가 준비되기 전까지 심장압박을 실시한다. 제세동기가 도착하면 쇼크 1회 후 이어서 다시 심장압박을 시행하고, 정맥로가 확보된 경우 우선 vasopressor(epinephrine or vasopressin)를 투여한다. 그 후에도 계속 심실세동 또는 맥박이 잡히지 않는 심실빈맥이 나타나면 항부정맥제(amiodarone or lidocaine) 투여를 고려한다.

**4) 예방** : 삽입형 제세동기(Implantable cardioverter defibrillator, ICD)
- ※ 급사 예방을 위한 ICD 적응증
  - ① (빨간 형광펜)**VT, VF로 인한 심정지 과거력**
  - ② 구조적 심질환이 있으면서 sustained VT
  - ③ 혈역학적으로 불안정한 VT, VF
  - ④ 다른 치료에 반응하지 않는 sustained VT 등

### 13. 각 차단 (Bundle branch block, BBB)

> 💡 각 차단은 **V₁(우심실)과 V₆(좌심실)를 짝지어** 봅니다.

**1) 우각차단 (Right bundle branch block, RBBB)**
- (1) 심전도 : V₁ → 마지막에 솟는 양상 (빨간 형광펜)**rSR' (M자 모양)** / V₆ → qRs, 깊고 넓은 S파(slurred S)
- (2) 원인
  - ① 구조적 심질환 없는 정상인에서도 흔히 관찰될 수 있으며, 이때는 경과관찰
  - ② 선천성 심질환(ASD, VSD), 후천성(판막질환, 허혈성 심질환)에서 동반

> 🖼 [그림필요 · p29] RBBB ECG (V₁ rSR'/M자, V₆ slurred S) — (미확보: 원서 p29 ECG strip 스캔 필요)

**2) 좌각차단 (Left bundle branch block, LBBB)**
- (1) 심전도 : V₁ → rS 또는 QS, 깊고 벌어진 S파 / V₆ → slurred R, notched R
- (2) 원인
  - ① 관상동맥질환(좌심실 기능 장애 동반), 고혈압성 심질환, 대동맥판막질환, 심근질환 등 구조적 심질환이 있는 환자에서 발생
  - ② 현재 증상이 없어도 구조적 심질환이 동반될 가능성이 있으므로 (노란 형광펜)**심초음파를 시행**

### 14. 디지탈리스 중독 (Digitalis intoxication)

**1) 증상** : 심전도만으로 진단할 수는 없고, (노란 형광펜)**Atrial fibrillation과 함께 Digitalis 복용력**이 있어 의심하게 됨. 위장관 증상(N/V), 시력 이상(blurred vision, 황색 시야), 정신·신경 증상(중추신경 증상)이 나타날 수 있음

> 🖼 [그림필요 · p30] 디지탈리스 효과 ECG ("Scooping" ST depression) — (미확보: 원서 p30 ECG strip 스캔 필요)

**2) 위험요인**
- ① 고령
- ② hypokalemia, hypomagnesemia, hypercalcemia
- ③ hypoxemia
- ④ renal insufficiency
- ⑤ acute myocardial infarction
- ⑥ chronic lung disease
- ⑦ hypothyroidism
- ⑧ quinidine, verapamil, amiodarone, propafenone 등을 같이 사용

**3) 검사**
- (1) (노란 형광펜)**혈청전해질(serum electrolyte)** : K⁺, Mg²⁺, Ca²⁺ level
- (2) BUN/Cr
- (3) (빨간 형광펜)**혈청 디곡신 농도(serum digoxin) : > 2ng/mL**
- (4) Thyroid function test
- (5) ECG

**4) 치료**
- (1) 디곡신 중단
- (2) 일시적 심장박동조율기(Temporary pacemaker) : 저혈압·부정맥·서맥 동반 시 사용
- (3) 부정맥 발생 시 (노란 형광펜)**베타차단제, lidocaine**
- (4) 저칼륨혈증이 동반된 경우 주의하며 KCl을 투여할 수 있음
- (5) Severe intoxication 시 : purified Fab fragment(digoxin-specific antibody)

> 💡 (빨간 볼드 밑줄)**Ca²⁺는 부정맥을 유발하므로 정맥주사 금기! DC cardioversion도 금기!**

### 15. 고칼륨혈증 (Hyperkalemia)

**1) 정의** : plasma K⁺ > 5.5mEq/L

**2) 심전도** (혈청 K⁺ 농도가 오를수록 단계적으로 변화)
- (1) 만성신부전 병력과 심전도상 (빨간 볼드 밑줄)**Tall T wave** 소견이면 고칼륨혈증을 의심하여 혈청전해질 검사를 통해 확인
- (2) low voltage of P waves : 7mEq/L
- (3) PR prolongation, disappearance of P waves, widening of QRS : > 8mEq/L
- (4) (노란 형광펜)**sine wave QRS-T pattern : > 10mEq/L** (→ VF로 진행 가능)

> 🖼 [그림필요 · p31] 혈청 K⁺ 농도(6.5 → 7.0 → 8.0 → 9.0 mEq/L)에 따른 ECG 변화 단계도 (tall T → P 소실/QRS 확대 → sine wave) — (미확보: 원서 p31 ECG strip 스캔 필요)

**3) 치료**
- ① Insulin + Glucose, β₂-agonist(Albuterol) : K를 세포 내로 이동시켜 일시적으로 효과
- ② Furosemide, Thiazide, (노란 형광펜)**양이온교환수지(Calcium polystyrene sulfonate; Kalimate)** : K를 몸 밖으로 배출
- ③ (노란 형광펜)**Ca²⁺ injection** : 세포 안정화. heart에서 tall T가 보이는 경우 → (노란 형광펜)**칼슘 글루코네이트(Calcium gluconate)** 주사

### 16. 심장성 급사를 잘 일으키는 3대 질환 (급사의 가족력)

> 💡 (빨간 볼드 밑줄)**급사의 가족력을 보이는 대표 3대 질환 = Brugada syndrome · Congenital long QT syndrome · HCMP**

#### 16-1. 브루가다증후군 (Brugada syndrome)

**1) 정의** : 특발성 심실세동(기질적 심질환이 동반되지 않으면서 심인성 급사가 나타나는 경우) 중, 심전도 소견상 (노란 형광펜)**RBBB 양상과 동시에 V₁~V₃ ST elevation**을 보이는 경우

**2) 원인**
- ① 상염색체 우성 유전
- ② Na 채널 유전자 돌연변이로 Na 전류가 감소하면서 심장 전도계 이상 → 회귀에 의한 심실 부정맥 유발

**3) 증상**
- ① 휴식·수면 시 실신
- ② 호흡 이상, 경련, 심계항진, 어지러움 등
- ③ 급사 (빠른 심실빈맥 또는 심실세동)

**4) 진단**
- ① 특징적 심전도 : (노란 형광펜)**V₁~V₃에서 ST elevation이 RBBB 형태**로 나타남
- ② 약물 유발 검사 : Na 채널 통로 억제제를 사용해 심전도 변화를 일으켜 잠재된 유전자 변이 추정
- ③ 급사의 가족력, 실신, 발열에 나타나는 호흡 이상

> 🖼 [그림필요 · p32] Brugada 심전도 3 type — Type 1 : Coved type ST elevation / Type 2 : Saddle-back type ST elevation / Type 3 : type 1·2 특징 + ST elevation < 2mm — (미확보: 원서 p32 ECG 도식 스캔 필요)

**5) 치료**
- ① 약물을 통한 치료는 어려움
- ② (빨간 볼드 밑줄)**심실세동이 발생한 경우, 제세동을 시행할 수 있는 삽입형 제세동기(ICD)를 삽입**

---

#### 16-2. 긴 QT 증후군 (Long QT syndrome)

**1) 정의** : 심실재분극의 지연으로 QT 간격이 길어지는 부정맥. polymorphic QRS complex와 QT 간격의 연장을 동반

**2) 종류**
- ① acquired long QT syndrome (pause dependent)
- ② congenital long QT syndrome (선천적인 비정상적 adrenergic 항진)

**3) 원인** (repolarization을 연장시키는 상황)
- ① bradyarrhythmia (3° AV block, VPB)
- ② 전해질 불균형 : 저칼륨혈증, 저칼슘혈증, 저마그네슘혈증 → K⁺·Ca²⁺ 통로에 영향
- ③ 약물 : 현저한 QT 연장 후 polymorphic VT 유발

**4) 심전도** : 현저한 QT interval 연장 후에 polymorphic VT

> 🖼 [그림필요 · p32] Long QT ECG (길어진 QT 간격 후 polymorphic VT/torsades로 이행) — (미확보: 원서 p32 ECG strip 스캔 필요)

**5) 치료 : QT interval을 짧게(HR ↑) 만들면 됨**
- (1) Acquired long QT syndrome
  - ① 원인 제거(전해질 교정, 약물 중단) : 대부분의 발작은 자연 종료
  - ② (빨간 형광펜)**Magnesium sulfate IV (TOC)** : 약물 유발성 TdP에 매우 효과적, Mg 정상이라도 투여
  - ③ Lidocaine
  - ④ β-agonist(isoproterenol) IV(주의 깊게 사용), temporary pacemaker → QT 짧게(HR ↑ 90~120회/min)
- (2) Congenital long QT syndrome
  - ① (빨간 형광펜)**β-blocker(propranolol, nadolol) : TOC**
  - ② Phenytoin : QT interval 단축 효과
  - ③ ICD

---

#### 16-3. 비대심근병증 (Hypertrophic cardiomyopathy, H-CMP)

**1) 주손상** : diastolic impairment → diastolic filling pressure 증가
- ① asymmetrical septal hypertrophy
- ② LV outflow obstruction (EF > 60%)
- ③ 자세에 따른 심잡음 변화
  - 심잡음 증가 : LV volume 감소 시 → Valsalva maneuver, sudden standing
  - ※ 일반적으로 Valsalva·sudden standing 시 "venous return ↓ → 심잡음 ↓"이 정상 기전이지만, (빨간 볼드 밑줄)**H-CMP와 승모판 탈출증(MVP)에서는 반대로 심잡음 ↑**
  - 심잡음 감소 : LV volume 증가 시 → Squatting

**2) 심잡음** : 제4심음, 수축기잡음(흉골 좌하연) [vs 승모판 탈출증은 apex]
- ※ 승모판 leaflet의 Systolic Anterior Movement(SAM) 현상으로 수축기에 MV가 LVOT로 빨려 들어가 AS와 유사한 심잡음을 형성하고, 같은 이유로 승모판이 잘 닫히지 않아 MR이 동반되는 경우가 흔함

**3) 증상** : (빨간 형광펜)**Dyspnea, anginal pain**, 부정맥, 돌연심장사

**4) 검사**
- ① 심전도 : LVH, nonspecific ST-T change, (빨간 형광펜)**giant T wave inversion**
- ② 심초음파 : asymmetrical septal hypertrophy, SAM(systolic anterior motion of MV)

> 🖼 [그림필요 · p34] H-CMP 12유도 심전도 (LVH, giant T wave inversion) — (미확보: 원서 p34 ECG strip 스캔 필요)

**5) 치료**
- ① 원칙 : (빨간 형광펜)**preload와 afterload를 증가시키는 방향**으로 치료 (CHF 치료와 반대 ; LV volume을 늘려 심잡음 ↓)
- ② (빨간 형광펜)**β-blocker**, non-DHP CCB(Verapamil, Diltiazem)
- ③ nifedipine은 금기 (arteriole을 확장시켜 BP를 떨어뜨리고 HR을 증가시킴)
- ④ (노란 형광펜)**제세동기 삽입(ICD)** : 고위험군에서
- ⑤ 수술 : 비후 중격의 surgical myectomy
- ⑥ 에탄올 주입술 (심실중격에 알코올 주입으로 괴사 유도)

### 17. Torsades de Pointes (TdP)

**1) 정의**
- (1) 심실빈맥의 특이 형태
- (2) QRS군이 주기적으로 높아졌다 낮아졌다 하는 일종의 polymorphic VT
- (3) (빨간 형광펜)**QT interval의 연장**을 동반하며 baseline을 중심으로 회전하는 모양

> 🖼 [그림필요 · p35] Torsades de Pointes ECG (QT 연장 + baseline을 중심으로 진폭이 변하며 회전하는 polymorphic VT) — (미확보: 원서 p35 ECG strip 스캔 필요)

**2) 유발 요인** (repolarization이 연장되는 상황 = QT interval이 연장되는 상황)
- (1) bradyarrhythmia (3° AV block, VPB)
- (2) 전해질 불균형 : 저칼륨혈증, 저칼슘혈증, 저마그네슘혈증 → K⁺·Ca²⁺ 통로에 영향
- (3) 약물 : K⁺ 통로 억제
  - ① class IA : quinidine, procainamide, disopyramide
  - ② class III : sotalol, amiodarone, ibutilide
  - ③ 기타 : phenothiazine, TCA, erythromycin, quinolone, ketoconazole, terfenadine 등
- (4) intracranial events : SAH

**3) 치료** (Long QT syndrome 치료에 준함 — QT를 짧게/HR ↑)
- (1) Acquired long QT syndrome
  - ① 원인 제거 : 대부분의 발작은 자연 종료. 전해질 이상 교정, 약물에 의해 유발된 경우 약물 중단 후 (빨간 형광펜)**overdrive pacing, MgSO₄ 투여**
  - ② (빨간 형광펜)**Magnesium sulfate IV (TOC)** : 약물 유발성 TdP에 매우 효과적, Mg 정상이라도 투여
  - ③ Lidocaine(class IB) : 항부정맥제 중 QT prolongation을 일으키지 않음
  - ④ Isoproterenol(β-agonist) IV(주의 깊게 사용), temporary pacemaker : HR를 90~120회/분으로 올려 QT를 단축시키고 VPC를 억제하여 원인 교정 시간을 벌 수 있음 (단, 허혈성 심질환 환자에선 금기!)
  - ⑤ ICD : life threatening arrhythmia를 겪은 모든 long QT syndrome 환자에게 추천
  - ※ (빨간 볼드 밑줄)**금기 약물 : quinidine(IA), procainamide(IA), amiodarone(III)** — 항부정맥제는 QT interval을 연장시키는 유발 약제이므로 금기!
- (2) Congenital long QT syndrome
  - ① (빨간 형광펜)**β-blocker(propranolol, nadolol) : TOC** / ※ 금기 : Isoproterenol(β-agonist)
  - ② Phenytoin(IB) : QT 단축 효과
  - ③ ICD (β-blocker 치료 실패 시)
  - ④ Cervicothoracic sympathectomy (부작용 : Horner syndrome)
  - ⑤ Overdrive pacing
