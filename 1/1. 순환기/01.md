# 순환기 (Cardiology)

## 심혈관계 진찰

> **강의 들어가며** — 순환기 첫 중단원입니다. 환자를 만나 ① 혈압을 재고 → ② 실신 같은 증상을 분류하고 → ③ 맥박·경정맥압을 보고 → ④ 청진으로 심음·잡음을 듣고 → ⑤ 심전도·심초음파로 확인하는, **진찰의 순서 그대로** 소단원을 배치했습니다. 시험은 이 단원에서 "어떤 소견 → 어떤 질환"을 묻는 매칭 문제가 압도적으로 많습니다.

### 01. 혈압 측정

**측정 방법에 따른 고혈압 진단 기준**

| 측정 방법 | 구분 | 수축기혈압(mmHg) | 이완기혈압(mmHg) |
|---|---|:---:|:---:|
| 진료실혈압 | | (빨간 볼드 밑줄)**≥ 140** | (빨간 볼드 밑줄)**≥ 90** |
| 24시간 활동혈압 | 일일(24시간) 평균 | ≥ 130 | ≥ 80 |
| | 주간 평균 | ≥ 135 | ≥ 85 |
| | 야간 평균 | ≥ 120 | ≥ 70 |
| 가정혈압 | | ≥ 135 | ≥ 85 |

> 💡 진료실혈압 기준만 140/90이고, 나머지(가정·주간활동)는 135/85, 24시간 평균은 130/80으로 묶어서 외우면 편합니다.

**1) 혈압계의 선택**
- 공기주머니 크기 : (노란 형광펜)**너비는 위팔 둘레의 37~50%, 길이는 위팔 둘레의 75~100%**가 적절
- 팔 둘레에 맞는 크기를 사용 : (빨간 형광펜)**너무 짧으면 실제보다 높게, 너무 길면 실제보다 낮게** 측정됨

**2) 측정 방법**
- 안정된 상태에서 측정 : (노란 형광펜)**5분간 안정**, 측정 (노란 형광펜)**30분 이내에는 흡연·알코올·카페인 제한**
- 반복 측정 : 1분 간격을 두고 적어도 2번 이상 측정

**3) 사지의 혈압**
- 반드시 양쪽 상지의 혈압을 모두 측정 (정상적으로 5~10mmHg 차이 가능)
  - cf. (빨간 형광펜)**Takayasu's arteritis : 좌우 상지 혈압 차 > 10mmHg**
- 하지는 상지보다 20mmHg 미만 정도 높게 측정됨
  - cf. (빨간 형광펜)**Coarctation of aorta : 상지 혈압 > 하지 혈압**

**4) 혈압이 실제와 다르게 측정되는 경우**
- 실제보다 **낮게** 측정 : 청진기를 세게 누름, 팔이 심장보다 높음, 공기주머니 길이가 김
- 실제보다 **높게** 측정 : (노란 형광펜)**동맥경화증(가성고혈압)**, 팔을 지지하지 않고 측정, 백의 고혈압

**5) 활동혈압/가정혈압 측정이 필요한 경우**
- (빨간 형광펜)**백의고혈압 (white coat hypertension)** : 진료실혈압은 높으나(≥140/90) 진료실 밖 혈압은 정상
- (빨간 형광펜)**가면고혈압 (masked hypertension)** : 진료실혈압은 정상이나 진료실 밖 혈압은 높음 (가정/주간활동 ≥135/85, 24시간 평균 ≥130/80)

### 02. 실신 (Syncope)

**1) 정의** : (노란 형광펜)**뇌혈류량 감소로 인한 일시적 의식 소실**

**2) 원인 및 감별진단** : 크게 신경심인성 실신 · 심혈관성 실신 · 상황 실신으로 구분
- 상황 실신 : 기침·배뇨·배변 등으로 유발되는 실신, (빨간 형광펜)**carotid sinus의 과민반응에 의한 실신 (Carotid sinus syndrome)**

> 🖼 [그림필요 · p4] **실신의 감별진단 흐름도** — 병력·이학적 검사 → ① 특이소견 없음→ 신경심인성 실신(약 50%) [재발 시 head-up tilt test] / ② 심질환 의심·신체검진 비정상 → 심장성 실신 [ECG·심초음파, 24h Holter monitoring(부정맥), electrophysiologic testing] / ③ 기립성 저혈압 → 신경학적 검진 정상 → 자율신경계 이상 / ④ 약물 복용력 확인, 신경학적 검진 이상 → 말초신경병증(당뇨·영양이상·amyloidosis) 또는 CNS 병변(multiple system atrophy) — (확보: assets/순환기/p003_syncope_algorithm.png)

### 03. 혈관미주신경성 실신 (Vasovagal / Vasodepressor / Neurocardiogenic syncope)

**1) 유발인자** : 덥고 사람 많고 답답한 환경(ex. 더운 여름 만원버스), 극심한 피로, 통증, 음주, 정신적 스트레스 등
> 💡 시험에서는 **오래 서 있을 때 증상이 발생**하는 증례로 자주 출제되었습니다.

**2) 기전** : (빨간 형광펜)**부교감신경(미주신경) 활성(서맥) + 교감신경 억제(혈관 이완)**
- 부교감신경(미주신경) 활성 → 심박수 감소
- 교감신경 억제 → 혈관 평활근이 제대로 수축하지 못함 → 혈압 감소

**3) 임상증상**
- 전구증상(주로 앉거나 서 있을 때 허약감·어지러움·구토·발한·두근거림) → (노란 형광펜)**cardiac slowing(서맥·저혈압)** → 실신
- 의식 소실 후 바로 정신이 돌아옴

**4) 검사** : 유발인자·전구증상 확인 + (빨간 형광펜)**기립경사검사(Head-up tilt test)**, 심전도
> 💡 정상인은 기립 시 다리로 혈액이 쏠려도 보상기전으로 혈압을 유지하지만, 혈관미주신경성 실신에서는 **혈압과 심박수가 함께 감소**합니다.

**5) 치료**
- 대증치료(예방 교육) : (노란 형광펜)**안심시키고, 실신을 일으키는 상황·자극을 피하도록 교육**
- 약물치료 : (노란 형광펜)**β-blocker, fludrocortisone, vasoconstricting agents, SSRI(fluoxetine)**
- 보존적 치료 : (노란 형광펜)**IV N/S**

### 04. 기립성 저혈압 (Postural / Orthostatic hypotension)

**1) 정의** : 자율신경반사 손상 또는 순환 혈액량 감소로, 서 있는 자세에서 유의한 혈압 강하가 나타나는 것
> 💡 **누운 자세에서 갑자기 일어설 때** 발생합니다.

**2) 원인**
- 약물 : 항고혈압제, 전립선비대증 치료제(α차단제), 항우울제(SSRI 등)
- Hypovolemic state : 급성/만성 체액용적고갈(volume depletion)
- PNS disorders : diabetic, nutritional, amyloid polyneuropathy
- CNS disorders : multisystem atrophy
- 오래 누워있는 자세나 무중력 상태
- 노화
- 기타 : 심장 펌프 기능부전(대동맥 협착, 심낭염/심근염, 부정맥), 부신 기능부전, 크롬친화세포종

**3) 진단**
- (빨간 볼드 밑줄)**기립혈압측정 : 똑바로 누워 5분 휴식 후, 기립 2~5분 내에 수축기혈압 ≥ 20mmHg 또는 이완기혈압 ≥ 10mmHg 저하**
- 머리기울임검사(head-up tilt test = tilt table test) : 일어설 때 특징적 반응 (3분 이내에 혈압 20/10mmHg 이상 감소)

**4) 치료**
- (노란 형광펜)**천천히 단계적으로 일어나기**, 일어나기 전 다리를 움직여 venous return 촉진
- (빨간 형광펜)**증상을 유발하는 약제 중단** : 이뇨제, 교감신경 차단제, nitrate계 협심증 치료제, α-아드레날린 길항제, SSRI 등
- 약물 치료 : Fludrocortisone, Midodrine(vasoconstrictor), Pyridostigmine, Yohimbine, Desmopressin
- 비약물 치료 : 압박스타킹 착용, 침대 머리 높이기, 수분·염분 섭취량 늘리기

### 05. 심혈관계 질환에 의한 실신

**1)** 주로 부정맥 등에 의해 (노란 형광펜)**심박출량이 갑자기 감소**하면서 발생
- 동성서맥(SSS), 3도 방실차단(complete AV block), 심실빈맥(VT) 등 부정맥
- (빨간 형광펜)**대동맥판막협착(AS)** 등 유출로 폐쇄, stroke volume이 충분히 늘지 않는 경우

**2)** (노란 형광펜)**전구증상이 짧고 갑자기 발생**

**3) 사례별 부정맥** : 동정지(sinus arrest)·동기능부전증후군(SSS), 방실전도차단(AV block), PSVT·AF·WPW syndrome, Torsades de pointes(QT 연장 — 약물에 의한 QT 연장, 저칼륨혈증 등)

**4) 강심제·승압제 (Inotropes / Vasopressors)**

- (노란 형광펜)**β₁ adrenergic = inotropic(수축력 ↑) / β₂ adrenergic = chronotropic**
- (용어) inotropes : 수축력 ↑ / chronotropics : HR ↑ / lusitropics : 심근 이완 ↑
- 수용체별 작용
  - α : (노란 형광펜)**말초혈관 수축 → 혈압 상승** (고용량 Dopamine, E / NE)
  - β₁ : 심장 수축 = CO·HR 상승
  - β₂ : 심장 외 근육 이완 = 저혈압 유발(혈관평활근 이완)·기관지 확장 작용

| 분류 | Drug | Principal mechanism | HR | MAP | CO | SVR |
|---|---|---|:---:|:---:|:---:|:---:|
| Inotropic agents (may be chronotropic) | Dobutamine | β₁-adrenergic | ↑↑ | ↑ | ↑ | ↓ |
| | Dopamine (low dose) | β₁-adrenergic : dopaminergic | ↑ | ↑ | ↑ | → |
| | Epinephrine (low dose) | β₁ and β₂-adrenergic : less α | ↑↑ | ↑ | ↑ | ↓ |
| Vasoconstrictors and inotropic agents | Dopamine (high dose) | α-adrenergic : less dopaminergic | ↑↑ | ↑↑ | ↑ | ↑↑ |
| | Epinephrine (high dose) | α-adrenergic : less β₁ and β₂ | ↑↑ | ↑↑ | ↑ | ↑↑ |
| | Norepinephrine | α-adrenergic : less β₁ and β₂ | ↑ | ↑↑ | ↑ | ↑↑ |

> 🖼 [그림필요 · p6] 위 표는 원서 도표를 옮긴 것 — 화살표 방향까지 동일한 원본 도표 참고 — (확보: assets/순환기/p005_inotrope_table.png)

- Dopamine : α, β₁ 수용체 작용 → 혈관 수축·혈압 상승
- Dobutamine : vasopressor가 아니라 (빨간 형광펜)**vasodilation 효과를 가진 inotropic** (심근 α₁·β₁·β₂ 작용 → 수축력·심박출량 증가)

### 06. 시진·촉진 — 유전질환 · 동맥압과 맥박 · 경정맥압

> 📌 원서에서는 이 부분의 소제목이 중단원과 똑같이 "심혈관계 진찰"로 되어 있습니다. 강의에서는 내용에 맞춰 **시진·촉진 파트**로 이름을 붙였습니다.

**1) 유전/환경적 인자에 의한 대표적 심질환**
- 다운증후군 : VSD, (노란 형광펜)**Atrioventricular Septal Defect(AVSD, ECD ; m/c)**, ASD
- 터너증후군 : (빨간 형광펜)**대동맥 축착(Coarctation of Aorta, CoA)**
- 마르판증후군 : (빨간 형광펜)**Aortic dissection, Aortic aneurysm**

**2) 동맥압과 맥박**

| 맥박 양상 | 연관 질환 |
|---|---|
| 큰 맥박 | Complete heart block, PDA, AV fistula, MR, VSD, AR |
| 작은 맥박 | MS, AS, Hypovolemia |
| 지연맥 (pulsus tardus) | (노란 형광펜)**AS** |
| 이단맥 (pulsus bisferiens) | AR(±AS), Hypertrophic cardiomyopathy |
| 중복맥 (dicrotic pulse) | Dilated cardiomyopathy |
| 교대맥 (pulsus alternans) | 심부전(S₃ gallop 동반), LV 수축기능의 심한 저하 |
| 기이맥 (pulsus paradoxus) | (빨간 형광펜)**심낭삼출·심낭압전**, 교착성 심낭염, 우심부전, 폐기종, 만성 천식 |

> 💡 (빨간 볼드 밑줄)**기이맥(pulsus paradoxus) = 흡기 시 수축기 혈압이 ≥ 10mmHg 감소** — '심낭압전'과 짝지어 출제됩니다.

**3) 경정맥압 (Jugular venous pressure, JVP)** → central venous pressure 반영
- 정상 소견 : 참고치 3~5cm. (노란 형광펜)**흉골각(sternal angle)으로부터 속목정맥(internal jugular vein) 박동 최고점까지의 수직거리**를 측정 (여기에 5를 더하면 우심방압)

> 🖼 [그림필요 · p7] JVP 측정 해부도 (SCM · external/internal jugular vein · clavicle · sternal angle · RA, 환자 상체 45°) — (미확보: 원서 p7 우측 삽화 스캔 필요)

- 비정상 소견
  - (빨간 형광펜)**Kussmaul's sign** : 흡기 시 경정맥압이 (정상은 하강해야 하는데) 오히려 증가 → 교착성 심막염, 제한성 심근병증, RV infarction
    > 💡 (빨간 볼드 밑줄)**심낭압전(cardiac tamponade)에서는 Kussmaul's sign이 나타나지 않는다!**
  - Hepatojugular reflux : 비스듬히 누운 환자의 RUQ(또는 중심부)를 10초간 압박 시 JVP 상승, 손을 떼면 3~4cm 급격히 감소 → 우심실부전 등

### 08. 심음 (Heart Sounds)

| 종류 | 소리 | 임상적 의의 |
|---|---|---|
| S₁ | 승모판(M₁)·삼첨판(T₁)이 닫히는 소리 | 증가 : (빨간 형광펜)**Mitral stenosis** / 감소 : (빨간 형광펜)**Mitral regurgitation** |
| S₂ | 대동맥판(A₂)·폐동맥판(P₂)이 닫히는 소리 (P₂가 조금 더 늦게 닫힘) | 아래 'S₂ 분열' 참조 |
| S₃ | ventricular filling rate/volume 증가 시 | 정상 : 소아·임신부·운동선수 / 비정상 : 심부전, ventricular function 저하(MR, AR) |
| S₄ | 심실 compliance(유순도) 저하 → 확장기말 심방수축에 의한 심실음(filling 저항 ↑ 의미) | (노란 형광펜)**AS, acute MR, systemic HTN, 허혈성 심질환, HCMP** |

**S₂ 분열 (Splitting of S₂)** — 시험 단골
- 정상(생리적) 분열 : 흡기 시 RV venous return ↑ → RV outflow ↑ → P₂가 늦게 닫힘
- (빨간 형광펜)**fixed splitting of S₂ → ASD**
- wide splitting : P₂ 지연(RBBB, PS, pul. HTN, pul. embolism) / A₂ 빨라짐(MR, VSD)
- paradoxical splitting : 호기 시 ↑·흡기 시 ↓ (A₂가 P₂보다 늦음) → (노란 형광펜)**LBBB(m/c), AS, systolic HTN**

### 09. 심잡음 (Murmur)

> 약어 — PE: Pulmonary embolism / RVF: Right ventricular failure / ASD: Atrial septal defect / LVF: Left ventricular failure / IHD: Ischemic heart disease

**1) 수축기 잡음 (Systolic murmur)**
- Holosystolic(Pansystolic) murmur : (빨간 형광펜)**VSD, MR, TR**
- Midsystolic(systolic ejection) murmur : (빨간 형광펜)**AS, PS**
- Late systolic murmur : (노란 형광펜)**MVP**

**2) 이완기 잡음 (Diastolic murmur)**
- Early diastolic murmur : (빨간 형광펜)**AR, PR** (with Pulmonary HTN)
- Mid-diastolic murmur : (빨간 형광펜)**MS, TS**

**3) 연속성 잡음 (Continuous murmur)** : (빨간 형광펜)**PDA**

> 🖼 [그림필요 · p9] 질환별 심잡음 파형 모식도 (S₁·S₂ 사이 도식으로 각 murmur 모양 + Fallot 4징 등) — (확보: assets/순환기/p013_murmur_shapes.png)

> 💡 대부분의 잡음은 정맥환류 감소 시 약해지지만, (빨간 볼드 밑줄)**HCMP와 MVP는 반대로 Valsalva 수기 시 잡음이 증가**합니다.

### 10. 무해성 심잡음 (Innocent murmur)

- **생리적 폐동맥분지협착 (Physiologic pulmonary branch stenosis murmur)** : 수개월 내 영유아에게 흔한 짧은 (노란 형광펜)**수축 중기 심잡음(midsystolic murmur)**으로, 복장뼈 왼쪽 가장자리 상부 또는 폐 말단 부위에서 들림
- **스틸잡음 (Still's murmur)** : 수축기에 중간 음조로 들리는 심잡음. 복장뼈 왼쪽 가장자리 중간~심첨부에서 잘 들리고, vibratory·squeaky한 양상. 누워 있을 때 잘 들리고 서 있으면 작아짐
- **정맥잡음 (Venous hum)** : 경정맥을 따라 들리는 연속성 잡음. (노란 형광펜)**자세를 바꾸면 소리가 변하고, 경정맥을 누르면 사라짐**

> 💡 (빨간 볼드 밑줄)**이완기에 들리거나 강도가 3도 이상인 잡음은 무해성 잡음이 아니다!** (→ 병적 잡음 의심)

### 11. 심전도 (ECG)

**1) 흉부유도(precordial lead) 위치**
- V₁ : 제4늑간 right sternal border / V₂ : 제4늑간 left sternal border
- V₃ : V₂와 V₄ 사이 / V₄ : (노란 형광펜)**제5늑간 left mid-clavicular line**
- V₅ : V₄와 같은 수평, left anterior axillary line / V₆ : V₄와 같은 수평, left mid-axillary line
- (빨간 형광펜)**V₄R : RV infarct(우심실 경색)를 가장 잘 반영** — Inferior MI에서 RV infarct 확인 / Dextrocardia(우심증)에서 reverse EKG

> 🖼 [그림필요 · p10] 사지유도(lead I·II·III, aVR·aVL·aVF) + 흉부유도 위치 도해 — (미확보: 원서 p10 우측 삽화 스캔 필요)

**2) 심전도 종합 해석표**

| 구분 | P파 (심방 탈분극) | QRS군 (심실 탈분극) | ST분절 / T파 / U파 (심실 재분극) |
|---|---|---|---|

| 해석 | 심방 문제(LAE, RAE), Sinus arrest, SA block | Narrow/Wide QRS(3칸), LVH, RVH | ST·T : (노란 형광펜)**ST 상승 = MI / ST 하강 = Ischemia·Angina**, Pericarditis / **Hyper K → hyper T** / U파 : Hypo K, Type 1 항부정맥제, TdP 가능성 |

|---|---|---|

### 12. 심초음파 (Echocardiography)

**1) 2D 심초음파**
- 심장 기능 : (노란 형광펜)**EF(ejection fraction) 측정**, wall motion abnormality
- 해부학적 구조 : 판막의 움직임 관찰, pericardial effusion 발생 등

**2) 도플러 심초음파 (Doppler echocardiography)** : 판막 협착·폐쇄부전 평가
- (빨간 볼드 밑줄)**valve gradient : ΔP = 4 × v²**

**3) 경식도 심초음파 (Transesophageal echocardiography, TEE)의 적응증** — 후방 구조를 잘 봄
- Inadequate TTE image (ex. LA appendage)
- Aortic disease : aortic dissection(thoracic)
- Source of embolism : (노란 형광펜)**atrial thrombi**, patent foramen ovale, aortic debris
- Intracardiac masses
- (빨간 형광펜)**Vegetation on valve : infective endocarditis**
- Valve prosthesis : 특히 mitral prosthesis의 abnormality 의심 시
- Intraop. mitral valve repair, septal myectomy

> 💡 TTE에서 잘 안 보이는 **LA thrombus·판막 vegetation** 등 후방 구조를 볼 때 TEE를 쓴다는 점을 기억하세요.
